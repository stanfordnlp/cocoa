You will be rating {batch_size} sets of dialogue responses. Please read the following instructions carefully.

In each question, you are given an excerpt of a conversation (the context) happened between a buyer and a seller negotiating the price of an item for sale, and two responses following the context. Your task is to decide which response is more appropriate in the context.

Mentions of prices are replaced by the symbol PRICE; you can assume the actual amount is reasonable. Symbols such as OFFER, ACCEPT, REJECT, QUIT denote corresponding actions. Given an action, you need to decide if it is reasonable given the context. For example, REJECT after agreeing to a deal (in the context) is not appropriate.

A good response should be grammatical, relevant to the context, and informative. For example, given the buyer's utterance
Buyer: Can you go lower?

Consider the following responses:
Seller A: It is in great condition!
Seller B: What about PRICE?
Seller C: The price is fair as it's practically brand new. However, if you can pick it up today I can do PRICE.
Response A is clearly bad because it is not answering the buyer's question. Both response B and C are okay. However, C is more specific and informative, therefore it is a better response.

In some cases, it is hard to decide which one is better. If they are equally appropriate, please select "Both are appropriate". If they are equally bad, for example, one is non-grammatical and one is irrelevant, please select "Both are NOT appropriate".

NOTE:
We use verification questions to identify abusers. In addition, each question is answered by multiple people. Outliers (e.g. randomly chosen answers) will get rejected.
